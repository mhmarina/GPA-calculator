# GPA-calculator
This app uses `java`, `maven` for build, `JavaFX` for GUI and `H2 DB` to locally store course and grade data.
## Prereqs
java, mvn
## Run Instructions
```
cd GPA-Calculator
```
```
mvn install
```
```
mvn javafx:run
```
